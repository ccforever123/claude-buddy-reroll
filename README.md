# Buddy 刷宠物工具

Claude Code Buddy 宠物刷取工具，支持交互式引导和中文界面。

## 工具列表

| 脚本 | 说明 |
|------|------|
| `buddy-reroll.js` | 刷宠物脚本，交互式生成并选择最优 Buddy |
| `buddy-auth.js` | Auth Token 替换工具 |

## 使用方法

### 刷宠物

```bash
bun buddy-reroll.js
```

### 替换 Auth Token

```bash
bun buddy-auth.js
```

## 功能特性

- 交互式中文引导界面，无需命令行参数
- 支持选择目标物种（18 种可选）
- 支持选择目标稀有度（普通/稀有/罕见/史诗/传说）
- 支持筛选闪光宠物
- 实时显示抽卡进度（各稀有度数量、百分比、进度条）
- 自动刷取 Top 10 候选，按质量排序
- 支持继续抽卡扩展候选（最多 9 次，累计 100 个）
- 交互式选择：输入序号选中，回车继续抽卡，0 退出

## 默认设置

- 目标稀有度：**传说**（5）
- 闪光：**是**（默认需要闪光）

## 稀有度概率

| 稀有度 | 中文名 | 概率 |
|--------|--------|------|
| common | 普通 | 60% |
| uncommon | 稀有 | 25% |
| rare | 罕见 | 10% |
| epic | 史诗 | 4% |
| legendary | 传说 | 1% |

闪光概率独立计算，为 1%。

## 使用流程

### 方式一：原版 Claude Code
1. 先运行一次 `claude auth` 完成登录授权
2. 再运行 `bun buddy-reroll.js` 刷宠物

### 方式二：中转版 / cc-switch
1. 直接运行 `bun buddy-reroll.js` 刷宠物

### 通用流程
1. 输入物种编号（1-18）或直接回车随机
2. 输入稀有度等级（1-5）
3. 选择是否需要闪光（y/n）
4. 等待刷出 Top 10 候选宠物
5. 输入序号选择，或直接回车继续抽卡
6. 重启 Claude Code，输入 `/buddy` 查看新宠物

## 配置文件

脚本会自动将宠物的 userID 写入 `~/.claude.json` 的 `userID` 字段，并删除原有的 companion 相关配置。
